# Android Deployer

Docker image that helps android devs to deploy there apps on different platforms.

### Supported Tools
- Appcenter
- Fastlane
- Firebase CLI

### Tools versions
- appcenter 
```
appcenter --version 
appcenter version 2.10.10
```
- Fastlane 
```
fastlane --version
fastlane 2.206.2
```
- Firebase CLI 
```
firebase --version 
11.0.1
```
